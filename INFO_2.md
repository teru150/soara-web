## 1. SOARAについて

### 1.1 基本情報
- **正式名称**: SOARA（ソアラ）- 高校生有志鳥人間プロジェクト
- **結成**: 2025年7月
- **メンバー**: 23〜24名（日米13校から参加）、平均年齢16.5歳
- **代表**: 入山輝大（開成高校2年）
- **目標**: 2026年7月 第48回鳥人間コンテスト滑空機部門出場、200m定常滑空達成
- **スローガン**: 「常識の向こうへ、碧空の彼方へ」/ "Beyond the Limit, Beyond the Blue Sky."
- **公式サイト**: https://soarahpa.com
- **SNS**: X @soara_hpa / Instagram @soara.hpa

### 1.2 設立経緯
代表の入山は当初、学校の部活動として鳥人間コンテストへの出場を目指したが、「前例がない」「資金調達を部活に認められない」等の理由で学校からの承認を得られず、学校の枠を超えて有志を募り結成。**高校生チームとして鳥人間コンテストに挑戦するのは史上初の試み**。

### 1.3 ミッション
1. **自分たちで作った飛行機を飛ばす夢を叶えること** - 困難を打ち破れることを証明
2. **ものづくりの面白さを同世代・次世代に伝えること** - 日本の理系選択者は約32%（OECD平均約50%）と低水準。高校という進路選択の重要な時期に航空・ものづくりの魅力を伝える

### 1.4 協力体制
- **東京理科大学 鳥人間サークル鳥科（TORICA）**: CFRP桁購入、技術指導、OBからのアドバイス
- **都立産業技術高専 Team ЯTR**: リブマスター製作技術の指導
- **京都工芸繊維大学 きっとFlyers**: 作業場見学、端部リブ処理などの知見
- **TOKYO INNOVATION BASE（TiB FAB）**: 現在の仮作業場、レーザーカッター等の設備利用

## 2. 機体仕様「SOARA-X3」

### 2.1 設計方針
東京理科大学の2024年度機体「Ray」を参考にした**堅実設計**。初年度は冒険的な設計を避け、確実に飛ぶ機体を作ることを最優先。

### 2.2 主翼
| 項目 | 仕様 |
|------|------|
| 形式 | 高翼（横安定重視） |
| 翼型 | GOE647（良好な失速特性で安全性担保） |
| 翼幅 | 18.8m |
| 翼弦長 | 1.4m（矩形翼 - リブ共通化で製作短縮） |
| 翼面積 | 26.32m² |
| 上反角 | 途中から4度 |
| 捻り下げ | なし（矩形翼のため） |
| 取付角 | +6度 |
| 重心位置 | 25.0%MAC |
| 桁構成 | 0-1-3-4-5-6番桁（2番桁は不使用 - 上反角確保のため） |

### 2.3 尾翼

**水平尾翼**
- 外形: 0.8m × 4.0m、面積: 3.2m²
- 翼型: NACA0012
- モーメントアーム: 3.5m
- 容積比: 0.30
- 取付角: -0.75度

**垂直尾翼**
- 外形: 0.8m × 1.4m、面積: 1.12m²
- 形式: 逆T字トップマウント
- 翼型: NACA0012
- モーメントアーム: 3.2m
- 容積比: 0.0072

### 2.4 運用・性能
| 項目 | 仕様 |
|------|------|
| 設計速度 | 約8.5m/s |
| 目標滑空比 | 15-20（理論値） |
| 目標飛行距離 | 200m以上 |
| 空虚重量 | 50kg |
| パイロット体重 | 55kg |
| 全備重量 | 105kg |
| ピッチ制御 | エレベーター（空力的フィードバック設計） |

## 3. CFRP桁について

### 3.1 購入元
東京理科大学 鳥人間サークルTORICA（有限会社スリーホープ製）

### 3.2 見積仕様（2023年9月29日付）
- 0桁〜6桁の7本セット
- 材料: P3252S-12（90°）、HRX350G125S（0°/±45°）

### 3.3 SOARAでの使用
- **使用する桁**: 0-1-3-4-5-6番桁（6本）
- **使用しない桁**: 2番桁
- **理由**: 6番桁を抜くと上反角が不足し横安定性が悪化するため、2番桁を抜いて対応
- **必要な作業**: 1番桁と3番桁を接合するかんざしを新規製作

### 3.4 かんざし製作の経緯
1. 12/27: かんざし試作開始
2. 12/28: 本番桁用製作 → GF巻いたら太くなって入らず失敗
3. 1/4-5: 鳥科のかんざしの上に3mm盛る方針に変更、調整
4. 1/6: **1.5G桁試験成功**

## 4. 翼製作の技術詳細

### 4.1 構造の分類
- **一次構造 (Primary Structure)**: 揚力を支える「背骨」。主桁（CFRPパイプ）とかんざし
- **二次構造 (Secondary Structure)**: 翼の形を作る「肉付け」。リブ、ストリンガー、プランク、後縁材、フィルム

### 4.2 各パーツの役割とSOARA仕様

#### ■ リブ (Rib)
| 項目 | SOARA仕様 |
|------|-----------|
| 役割 | 翼の断面形状（翼型）を維持する |
| 材料 | スタイロフォーム |
| 厚さ | 10mm |
| 切り出し | 重力式スライサーでスライス → リブ切り台でホットワイヤーカット |
| マスター | MDFレーザーカット（TiB FABで製作） |
| 分割構造 | 翼弦1.4mがレーザーカッターに入らないこと、および桁のフランジ（出っ張り）を回避するため、桁穴を境に「前縁側」と「後縁側」に分割して製作 |
| 接合 | 桁穴の周囲を六角形の1mm厚バルサプレートで両面から挟み込み、接合・補強・位置合わせを同時に行う |
| 端部リブ | リブ3枚重ね（きっとFlyersから学んだ手法） |

#### ■ ストリンガー (Stringer)
- **役割**: リブ間での外皮（フィルム等）の凹みを防ぐ
- **SOARA仕様**: 3×5mmヒノキ角材
- **重要**: 断面の長い方（5mm）を翼表面に対して垂直に立てて配置し、曲げ剛性を最大化する。プランク端から30mm内側で止めることで、フィルムへの移行を滑らかにする
- **接着**: スチのり（発泡スチロール用接着剤）

#### ■ プランク (Plank)
- **役割**: 前縁（Leading Edge）の形状を正確に保ち、層流を維持する
- **SOARA仕様**: スチレンペーパー（スタイロフォームも可能だが、SOARAではスチレンペーパー採用）

#### ■ リブキャップ (Rib Cap)
| 項目 | SOARA仕様 |
|------|-----------|
| 役割 | リブの縁を補強し、フィルムの張力から守る |
| 材料 | ヒノキ材（通常はバルサだが、20mmで売っていて精度出しやすいため採用） |
| 注意点 | 後縁ギリギリまで伸ばさない |
| 理由 | 木材は厚みがあり、リブキャップの上に後縁材が重なると段差ができ気流剥離の原因になる |
| 対策 | 先に後縁材を貼り、リブキャップは後縁材に直交する形で接合（重なり防止）。下面は主桁まで伸ばし、リブを挟み込む「サンドイッチ構造」にする |

#### ■ 後縁材 (Trailing Edge)
- **役割**: 翼の最後尾を鋭くし、空気抵抗を減らす
- **SOARA仕様**: ヒノキとスタイロの複合材。リブの後縁20mmを上下から1mmヒノキで挟み込み、リブ間には斜めにスライスしたスタイロを挿入する

### 4.3 接着剤の使い分け
| 用途 | 接着剤 | 理由 |
|------|--------|------|
| ストリンガー-リブ | スチのり | スタイロを溶かさない、軽い、柔軟性あり |
| リブ同士（端部リブ等） | スチのり | 同上 |
| かんざし製作（GF巻き） | エポキシ | 構造強度が必要 |
| 桁とリブの接合部 | エポキシ | 荷重伝達部分 |
| CFRP/GFRP全般 | エポキシ | FRP用 |

**注意**: エポキシは溶剤成分でスタイロフォームを溶かす可能性があるため、スタイロに直接使用しない

### 4.4 製作フロー：オン・スパ同時接着
桁の表面が荒く、フランジがあるため、SOARAでは**「リブの結合」「迎角合わせ」「桁への固定」を一つの工程で同時に行う**

1. **ドライフィット**: 接着剤なしで前後リブを桁にセットし、迎角保持ジグに乗せて隙間がないか確認
2. **接着剤塗布**: リブ接合面、バルサプレート、桁の接触面にエポキシ等を塗布
3. **同時圧着**: 迎角保持ジグにリブを押し付けながら、前後リブをバルサプレートでサンドイッチしてクランプ固定

## 5. 参考資料: チームあざみ野 2022年秋交流会資料

### 5.1 機体「Revenant Plus ARG-2」
- 2022年大会で533.58m（大会新記録）を達成
- SOARAと同じ18.8mスパン、GOE647翼型

### 5.2 設計思想
- 「m/円の最大化」- 廃棄桁を活用したコスト効率
- 「工業化」- 誰でも作れる設計（専門家依存を避ける）

### 5.3 翼製作の要点（清水総一郎氏）
- **リブ間隔 > 複雑な補強**: 多くのリブを使う方がシンプルで効果的
- **後縁材**: 最も目立つ品質指標、丁寧な処理が必要
- **ハード後縁 vs ソフト後縁**: 滑空機・低たわみ翼にはハード後縁が適する

### 5.4 フライト戦略
- フライトシム中心のパイロット訓練（6ヶ月間、毎日30分〜2時間）
- 設計者から無線で指示を出す方式
- 最小限のダイブ、設計速度維持、追い風活用

### 5.5 電装システム
- エアデータセンサー（対気速度、AoA、AoS）
- 速度ベースの音声フィードバック（300/500/700Hz）
- 高度計測: 超音波、気圧、GPS の3方式
